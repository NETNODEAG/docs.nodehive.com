# NodeHive Frontend Documentation

## Official starter template for NextJS

https://github.com/NETNODEAG/nodehive-nextjs-starter

```bash
npx create-next-app frontend.example.ch -e https://github.com/NETNODEAG/nodehive-nextjs-starter/tree/main
```

### Example .env

```
  # Required for NodeHive
  NEXT_PUBLIC_DRUPAL_REST_BASE_URL="https://yourinstance.nodehive.app"
  NEXT_PUBLIC_DRUPAL_BASE_URL="https://yourinstance.nodehive.app"
  NEXT_IMAGE_DOMAIN="yourinstance.nodehive.app"
  NEXT_PUBLIC_NODEHIVE_SPACE_ID=1
  NODEHIVE_STARTPAGE_SLUG=/node/1
  NODEHIVE_DEFAULT_LANGUAGE=en
```

## Custom frontend

You can use any frontend technology you like. The NodeHive backend offers a range of apis you need to get familiar.
