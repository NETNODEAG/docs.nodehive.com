## What is it?
The content planner is an integral part of NodeHive Headless CMS and brings a lot content planning powers.

## How to configure it

You have to have "NodeHive Administrator" role to configure the content planner.

1. Go to /admin/config/workflow/workflows/manage/editorial
2. Under "This workflow applies to:" > "Content types" > "Select" the content types you want to apply the workflow to and save
3. Go to the content type config page (Example: /admin/structure/types/manage/page) and enable the schedule settings
4. Now content planner is enabled for the content type /admin/content-calendar
5. Go to /admin/content-calendar/settings and configure content planner functionality itslef
