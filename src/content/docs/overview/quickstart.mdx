To get up and running with your NodeHive project, you need:

- A NodeHive backend instance
- A javascript frontend like NextJS

:::note
NodeHive Headless CMS is open source and provides all you need to start your headless project.
We also offer a hosted SaaS version available through www.nodehive.com.
:::

## Start local development

### 1. Install NodeHive backend locally

- Follow the instructions on https://github.com/NETNODEAG/nodehive-headless-cms-ce
- [Video Tutorial](https://www.youtube.com/watch?v=BrL3uH73CaA&list=PLx8ET0RIaWG2NcK6TiM7fC3TOOkejYNzF&index=12&)

### 2. Install NodeHive Starter Frontend

- Follow the instructions on https://github.com/NETNODEAG/nodehive-nextjs-minimal-starter
- [Video Tutorial](https://www.youtube.com/watch?v=zXmCDxb-tBE&list=PLx8ET0RIaWG2NcK6TiM7fC3TOOkejYNzF)
